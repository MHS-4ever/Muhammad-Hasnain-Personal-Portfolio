Scramble Math

  • Tech Stack: Unity, C#, UI Toolkit, Audio System, Timer System, Difficulty Scaling

Developed an interactive educational game designed for kids to enhance their arithmetic skills. The game generates random equations (e.g., 2+2=?) with either a missing operator or digit, and players must choose the correct answer from a set of options within a given time frame.

Integrated a dynamic scoring system where players earn one point for each correct answer. The game also tracks the highest score, motivating players to improve their performance and achieve new records.

The game's difficulty increases after a set number of correct answers, introducing more complex equations and reducing the time available to answer. This progressive difficulty system ensures that kids are constantly challenged while learning at their own pace.

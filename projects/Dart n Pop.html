Dart n Pop

  • Tech Stack: Unity, C#, Sprite Renderer, Animator, UI Toolkit, Particle Effects

Developed an engaging and interactive kids' fun game where balloons move in a circular pattern across the screen, and players must pop them using a dart launched from the bottom of the screen.

Implemented precise click mechanics, enabling players to release the dart with a simple screen tap. If the dart successfully hits a balloon, it pops, and points are added to the score. Missing a balloon results in a game-over scenario.

Integrated a high-score tracking system to encourage replayability and competitiveness. The game dynamically increases difficulty after a set number of successful pops by increasing the balloons' spin speed and adding more balloons to the screen.
